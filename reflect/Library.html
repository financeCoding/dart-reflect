Class Library

Represents a Dart library.

Constructors

Code new Library(String url) #

Wraps the library with the specified URL, loading it if it doesn't exist. url may be absolute or relative to the working directory. Throws an exception if the library doesn't exist and can't be loaded.

Library(String url) : this.url = url {
  _handle = _init(
    new Uri(scheme:"file", path:"${new Directory.current().path}/").resolve(url).toString(),
    () => new File(url).readAsTextSync());
}

Methods

Code Class getClass(String name) #

Returns a Class wrapping the named class. Throws an exception if the class doesn't exist.

Class getClass(String name) => new Class._internal(this, name);

Code Callable method(String name) #

Returns a Callable wrapping the named top-level function. Does not verify that the function exists.

Callable method(String name) => new Callable.fromHandler(
    (args) => _invoke(_handle, name, args),
    "Function $name");
